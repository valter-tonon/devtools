## 🚀 Começando

Após clonar o projeto basta rodar o comando:
```
docker-compose up -d
```
Acessar em [localhost:3000](http://localhost:3000)

### 🔧 Acesso CLI

Para acessar a linha de comando basta rodar o comando:
```
docker exec -it app-dev sh
```
